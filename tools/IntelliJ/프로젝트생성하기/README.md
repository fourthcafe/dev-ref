|ECLIPSE	|INTELLIJ
|:---|:---
|Workspace(Contains one or more projects)	|Project
|Project	|Module
|Project-specific JRE	|Module SDK
|User Library	|Global Library
|Classpath Variable |Path Variable
|Project Dependency	|Module Dependency
|Library	|Module Library
